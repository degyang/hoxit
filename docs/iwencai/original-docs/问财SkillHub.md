Skills：hithink-zhishu-query

## Agent和CLI用户
### CLI用户
在终端中运行以下命令以安装 SkillHub CLI：
```bash
curl -fsSL https://www.iwencai.com/skillhub/static/0.0.4/download_and_install.sh | bash -s
```
安装 CLI 后，运行以下命令安装技能
```bash
iwencai-skillhub-cli install report-search
```
检查并配置环境变量
```bash
curl -fsSL https://www.iwencai.com/skillhub/static/0.0.4/setup_iwencai_env.sh | bash -s -- --IWENCAI_API_KEY=your-api-key --IWENCAI_BASE_URL=https://openapi.iwencai.com
```
### Agent用户
将以下 Prompt 复制到您的 AI 助手中，即可完成技能安装。
```prompt
请先检查是否已安装Iwencai SkillHub商店，若未安装，请根据 `https://www.iwencai.com/skillhub/static/0.0.4/download_and_install.sh` 安装Iwencai SkillHub商店（仅安装 CLI），然后安装 `Skills-Name` 技能。

若已安装，则直接安装 `Skills-Name` 技能。

安装后，请确保 shell profile 中存在以下环境变量：

IWENCAI_BASE_URL=https://openapi.iwencai.com
IWENCAI_API_KEY=your-api-key
```
